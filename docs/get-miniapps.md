# 📄 获取小程序挂载列表 Query 参数 (Get MiniApps Query)

获取账号可用的挂载小程序列表。常用于抖音、视频号等平台在视频播放页面挂载互动组件。

> [!IMPORTANT]
> **挂载前提**：只有特定类型的账号（如企业号、认证号）或满足粉丝门槛的账号才具备小程序挂载权限。

## 1. 触发场景 (Trigger)

- **意图辨析**：用户希望在内容中添加导流入口或互动组件。
- **典型提示词**：
  - “这个视频需要挂载一个小程序”
  - “搜索名为‘抽奖助手’的小程序”

## 2. 交互协议 (Interactive Protocol)

1. **权限预判**：若账号 `status` 正常但小程序列表为空，主动询问用户是否已在平台后台开通挂载权限。
2. **精准匹配**：提供关键词 `keyword` 帮助用户从大量可用小程序中快速定位目标。
3. **透传原则**：获取选定小程序数据后，将其详情及 `raw` 对象整体填入发布表单。

## 3. 参数定义 (Parameters)

| 字段名 | 类型 | 必填 | 默认值 | 描述 |
| :--- | :--- | :--- | :--- | :--- |
| **`action`** | `string` | **是** | `miniapps` | 固定操作码。 |
| **`account_id`** | `string` | **是** | - | 蚁小二账号 ID (`platformAccountId`)。 |
| `keyword` | `string` | 否 | - | 小程序名称搜索关键词。 |

## 4. 执行指令示例 (Command)

```bash
node scripts/api.ts --payload='{"action":"miniapps","account_id":"67fb2f1735eeb3cf31db3d65","keyword":"抽奖"}'
```

## 5. 常见问题排查 (Troubleshooting)

| 现象 | 可能原因 | 处理建议 |
| :--- | :--- | :--- |
| **找不到特定小程序** | 该小程序未在平台管理后台通过关联。 | 引导用户先前往平台后台（如抖音创作园地）完成小程序绑定。 |
| **挂载后点击无效** | 小程序参数、路径或 `raw` 数据有误。 | 核实 `raw` 数据的完整性，不建议手动拼湊 appId。 |

---
> [!TIP]
> **转化建议**：小程序挂载建议选择与视频内容强相关的工具，以提升用户点击转化率。
