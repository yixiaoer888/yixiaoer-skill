# 📄 获取同步发布应用 Query 参数 (Get Sync Apps Query)

获取在发布当前作品时，可同时同步到的其他关联平台或应用（例如：抖音发布时可勾选同步到今日头条、西瓜视频）。

> [!NOTE]
> **多端同步逻辑**：同步发布是平台内部的链路，不消耗蚁小二额外的发布点数，但受平台规则限制（如必须是实名账号）。

## 1. 触发场景 (Trigger)

- **意图辨析**：用户希望一键实现平台内的多端联动。
- **典型提示词**：
  - “我发抖音的时候，能顺便同步到头条吗？”
  - “看看这个号支持同步到哪些应用”

## 2. 交互协议 (Interactive Protocol)

1. **权限前置**：必须先调此接口确认“同步开关”是否可用。
2. **状态记录**：记录返回的 `raw` 数据，若用户同意同步，则在 `contentPublishForm` 的对应同步字段中填入该对象。

## 3. 参数定义 (Parameters)

| 字段名 | 类型 | 必填 | 默认值 | 描述 |
| :--- | :--- | :--- | :--- | :--- |
| **`action`** | `string` | **是** | `syncapps` | 固定操作码。 |
| **`account_id`** | `string` | **是** | - | 蚁小二账号 ID (`platformAccountId`)。 |

## 4. 执行指令示例 (Command)

```bash
node scripts/api.ts --payload='{"action":"syncapps","account_id":"67fb2f1735eeb3cf31db3d65"}'
```

## 5. 常见问题排查 (Troubleshooting)

| 现象 | 可能原因 | 处理建议 |
| :--- | :--- | :--- |
| **列表为空** | 账号未绑定第三方同步应用，或平台暂不支持。 | 引导用户在平台手机端手动完成一次应用授权绑定。 |
| **同步失败** | 账号在同步目标端的状态异常。 | 检查同步目标的登录状态是否依然有效。 |

---
> [!CAUTION]
> **数据透传禁令**：如果在获取时 `raw` 字段有值，发布表单中**必须**完整保留并透传整个对象。严禁只通过 Boolean 值控制开关。
